## What I Built

A collaborative document editor. Think Google Docs but scoped down to what actually matters for proving I can ship a full-stack product under pressure.

You can create documents, write in them with real formatting, upload files, share docs with other users, and everything persists in a real database.

## Quick Start

```bash
git clone https://github.com/shauryaojaswa/ajaia-docs.git
cd ajaia-docs
npm install
npx prisma db push
npx prisma db seed
npm run dev
